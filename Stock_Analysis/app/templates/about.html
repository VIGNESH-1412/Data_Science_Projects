System Architecture

About the Project

StockPulse Stock Intelligence is a full-stack integration designed to bring machine learning out of experimental Jupyter Notebooks and into a high-fidelity interactive dashboard environment.

Nifty 50 Dataset

The underlying ML model was trained on the Indian Stock Market Data corpus. It consists of 59,071 daily trading records spanning from April 2017 to January 2022. It covers 50 major public constituents (e.g., ADANIPORTS, WIPRO, RELIANCE, SBIN).

Random Forest Regressor

We chose an ensemble Random Forest Regressor model. By training 100 decision trees on random data subsets, the model learns the highly structured correlations between a stock's daily Open, High, Low, Adjusted Close values, and its final trade Close price.

Execution Pipeline

1. Client Simulation Variable Tweaks

Users select a constituent on the Dashboard. Form fields autofill with the last recorded stock data from the CSV. Users adjust sliders to simulate specific price shocks.

2. API Serialization

Payloads (Open, High, Low, Adj Close, Volume, Company Name) serialize to JSON and post to Flask API endpoint `/api/predict`.

3. Joblib AI Inference

Flask loads the pre-trained `Stock_Market_Analysis.joblib` model at startup. When invoked, it executes python inference, returning a single predicted floating point value representing the closing price.

4. SQLite Logging & UI Render

Predictions log to local SQLite registers for history auditing. The frontend renders glowing results cards, trend directions, and dynamic error limits.
